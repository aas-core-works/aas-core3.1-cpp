Detailed Description

Represent a path to some value.

This is a path akin to C++ expressions. It is not to be confused with different paths used in the specification. This path class is meant to help with reporting. For example, we can use this path to let the user know when there is a verification error in a model which can concern instances, but also properties and items in the lists.
