Detailed Description

A property is a data element that has a multi-language value.

Constraint AASd-012:
If both the IMultiLanguageProperty::value and the IMultiLanguageProperty::value_id are present then for each string in a specific language the meaning must be the same as specified in IMultiLanguageProperty::value_id.
