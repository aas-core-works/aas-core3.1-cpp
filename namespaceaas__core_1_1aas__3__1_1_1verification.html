◆ MatchesXsAnyUri()

bool aas_core::aas_3_1::verification::MatchesXsAnyUri ( const std::wstring &  text)

Check that text conforms to the pattern of an xs:anyURI.

See: https://www.w3.org/TR/xmlschema-2/#anyURI and https://datatracker.ietf.org/doc/html/rfc2396 and https://datatracker.ietf.org/doc/html/rfc2732

Note, that version 1.0 of the XSD specification defines xs:anyURI as "defined by RFC 2396, as amended by RFC 2732". Therefore, we use a pattern here that implements the amendments of RFC 2732. This should not be confused with matches_RFC_2396, which does not include those amendments and is used in different parts of the specification.
