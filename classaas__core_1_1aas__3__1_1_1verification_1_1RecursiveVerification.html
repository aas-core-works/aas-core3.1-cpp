Detailed Description

Verify that the instance conforms to the meta-model constraints.

Also verify recursively all the instances referenced from the given instance.

Range-based loops should fit the vast majority of the use cases:

std::shared_ptr<types::Environment> env = ...;
for (const Error& error : RecursiveVerification(env)) {
report_somehow(error);
}

We use const references to shared pointers here for efficiency. Since we do not make a copy of that shared pointer, it is very important that the given shared pointer outlives the verification, lest cause undefined behavior. See these StackOverflow questions:
